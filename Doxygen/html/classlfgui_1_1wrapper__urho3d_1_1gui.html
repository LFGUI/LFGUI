Detailed Description

The LFGUI wrapper for Urho3D (http://urho3d.org/ or http://urho3d.github.io/). It will automatically subscribe to all events it needs and display itself on the screen.

Example usage:

class SampleApplication : public Application
{
public:
std::unique_ptr<lfgui::wrapper_urho3d::gui> gui;
...
virtual void Start()
{
// create our LFGUI instance and set up the example widgets
gui.reset(new lfgui::wrapper_urho3d::gui(GetContext()));
setup_sample_gui(gui.get());
